# IoT for industry-opencv
Opencv object detection with opencv and raspberry pi

Used:
  - Raspberry PI 4 4GB
  - Raspberry PI camera
  - LED 
  - Resistor


Installation of Open CV:

Enablea camera
  pi@raspberrypi:~ $ sudo raspi-config
